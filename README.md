# ProperProc
Sistema de gestión web para el Bazar Plásticos y Envases Descartables "NENO". Desarrollado con ASP.NET Core MVC + API REST, SQL Server y SignalR. Cubre 6 módulos: Administración, Configuración, Ventas, Compras, Caja y Reportes con dashboard en tiempo real. Proyecto de grado – UDABOL 2026.
